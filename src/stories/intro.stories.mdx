# Fresh-Designs

Fresh-Designs is a collection of beautiful reusable react JS components.

## Install

Add Fresh-Designs to your project.

`yarn add fresh-designs`

-or-

`npm install --save fresh-designs`

### **Use**

Import components you want into your UI

`import { Card} from ‘fresh-designs’;`

and use them like so

```
import { Card} from ‘fresh-designs’;

const example = () => (
    <div>
        <Card
            title="Fresh-Designs"
            subtitle="A collection of React JS components" />
    </div>
)
```
